# 🌤 Live Weather Application

A responsive Weather Application built using **HTML, CSS, and JavaScript**. It provides real-time weather information using the OpenWeatherMap API.

## 🚀 Live Demo

🔗 https://bremmi-weather-appilication.netlify.app

## ✨ Features

- 🔍 Search weather by city name
- 📍 Current location weather
- 🌡️ Current temperature
- 💧 Humidity
- 🌬️ Wind speed
- 🌡️ Feels like temperature
- 🧭 Atmospheric pressure
- 📅 5-Day Weather Forecast
- 📱 Responsive Design
- ⚠️ Error handling for invalid city names

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
